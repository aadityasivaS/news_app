# news_app

A news application with multiple sources of news developed in flutter.

## Getting Started

This project is a News application with multiple sources. The API used is from "https://newsapi.org".
Get API Key from "https://newsapi.org"

## Get dependencies

flutter pub get

## Build and Run

flutter run
